# 赛道规格图标说明

## 需要准备的SVG图标文件

请准备 **1个** SVG图标文件放置在此目录下：

### spec-icon.svg - 赛道规格图标
- 用途：用于所有赛道规格项（全长、海拔提升、弯道数量、弯道类型）
- 建议图标：赛车、赛道、方向盘、旗帜等通用竞速元素

## 图标设计要求

### 尺寸规格
- 推荐尺寸：32x32px（桌面端）
- 移动端会自动缩放到 28x28px
- SVG格式，矢量图形

### 颜色方案
- 建议使用红色主题色：#CD202B 或 #ff4757
- 也可以使用白色/灰色，系统会自动适配
- 确保在深色背景上清晰可见

### 风格建议
- 简洁、现代的线条图标
- 统一的设计风格
- 清晰的轮廓，适合小尺寸显示

## 文件放置
将准备好的SVG文件直接放在此目录下即可自动生效。
