# 合作伙伴 Logo 文件说明

## 目录结构
将所有合作伙伴的 logo 文件放在此目录下。

## Logo 文件要求

### 文件格式
- 推荐格式：PNG（支持透明背景）
- 备选格式：SVG、JPG

### 文件尺寸
- 最大宽度：300px - 500px
- 最大高度：100px - 150px
- 文件大小：建议小于 100KB

### 设计建议
- 使用透明背景（PNG）
- Logo 应为浅色或白色版本（因为背景为深色）
- 如果只有深色 logo，系统会自动应用滤镜处理

## 注意事项

1. **自动隐藏功能**：如果 logo 文件不存在或加载失败，图片会自动隐藏，只显示文字名称
2. **灰度处理**：所有 logo 会自动应用灰度滤镜和亮度调整，以适应深色背景
3. **响应式**：Logo 会自动缩放以适应不同屏幕尺寸

## Logo 添加后的效果

每个合作伙伴卡片将显示：
- 上方：Logo 图片（灰度处理）
- 下方：合作伙伴名称（中英文）

卡片采用网格布局，自动适应不同数量的合作伙伴。
