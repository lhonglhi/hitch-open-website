# 赛道图片文件夹

此文件夹存放赛程弹窗中显示的赛道图片。

## 图片命名规范

根据 `data/timeline.json` 中的赛程配置，需要以下图片：

### 2025 赛季

1. **oval-track.png** - R1 仿真椭圆赛道
   - 推荐尺寸：1200x800px 或更大
   - 格式：PNG/JPG
   - 说明：第一站基础赛道测试

2. **formula-track.png** - R2 仿真方程式赛道
   - 推荐尺寸：1200x800px 或更大
   - 格式：PNG/JPG
   - 说明：第二站复杂高速赛道

3. **tianmen-track.png** - R3 天门山决赛赛道
   - 推荐尺寸：1200x800px 或更大
   - 格式：PNG/JPG
   - 说明：天门山99道弯实体赛道

### 2026 赛季

4. **upcoming-track.png** - 2026赛季预告
   - 推荐尺寸：1200x800px 或更大
   - 格式：PNG/JPG
   - 说明：全新赛季预告图或概念图

## 图片要求

- **格式**：PNG（推荐）或 JPG
- **尺寸**：最小 1200x800px，建议 1600x1000px 以上
- **比例**：16:9 或 3:2
- **文件大小**：建议单个文件 < 500KB（优化后）
- **内容**：清晰的赛道俯视图或全景图
