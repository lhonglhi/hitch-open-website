# 占位图制作指南

## 🎨 什么时候使用占位图？

当某个学校的官方logo暂时无法获取时，使用占位图（placeholder.png）。

## 📐 占位图设计建议

### 方案1：简单圆形图标
```
┌─────────────┐
│             │
│     ⚪      │  灰色圆形
│             │  + 学校首字母
│             │
└─────────────┘
```

### 方案2：统一徽章样式
```
┌─────────────┐
│             │
│    🏫       │  学校图标
│   [学校]    │  + 文字
│             │
└─────────────┘
```

### 方案3：纯文字
```
┌─────────────┐
│             │
│   清华大学   │  学校名称
│      or     │  或英文缩写
│    THU      │
│             │
└─────────────┘
```

## 🎯 推荐规格

- **尺寸**: 200x200px
- **格式**: PNG（透明背景）
- **颜色**: 白色或浅灰色（#CCCCCC）
- **字体**: 简洁无衬线字体
- **背景**: 透明或深色（#1a1a1a）

## 🛠️ 在线制作工具

1. **Canva** - https://www.canva.com
   - 选择正方形画布（200x200）
   - 使用圆形/文字工具
   - 导出为PNG

2. **Figma** - https://www.figma.com
   - 专业设计工具
   - 支持矢量图形

3. **Photopea** - https://www.photopea.com
   - 免费在线PS
   - 支持透明背景

## 💡 快速方案

如果时间紧迫，可以：
1. 使用纯色圆形 + 学校名称首字母
2. 统一灰色调，保持视觉一致性
3. 等待后续替换为官方logo

## 📝 示例代码（SVG）

```svg
<svg width="200" height="200" xmlns="http://www.w3.org/2000/svg">
  <!-- 背景圆形 -->
  <circle cx="100" cy="100" r="80" fill="none" stroke="#CCCCCC" stroke-width="2"/>

  <!-- 学校首字母 -->
  <text x="100" y="120"
        font-family="Arial, sans-serif"
        font-size="60"
        fill="#CCCCCC"
        text-anchor="middle">清</text>
</svg>
```

保存为 placeholder.png 或使用在线工具转换。
